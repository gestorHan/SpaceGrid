# SpaceGrid
Juego en opengl sobre una nave que se mueve en una cauadricula, echo para la clase de Computacion Grafica


# Compilar usando:

`g++ *.cpp   -lGL -std=c++11 -Wall -lGL -lGLU -lglut -lGLEW -lsoil2-debug -lglfw -ldl   -o a.out`
